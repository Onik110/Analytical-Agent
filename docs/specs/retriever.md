# Спецификация: Retriever / Query Generator

## Источники данных

| Источник | Тип | Описание |
|----------|-----|----------|
| **SYSTEM_PROMPT** | Config | Схема базы 1С + правила генерации (~500 токенов) |
| **Sanitized Query** | Input | Запрос пользователя после очистки ФИО |
| **Date Range** | Input (opt) | Кортеж `(start_date, end_date)` для параметров &нач, &кон |
| **Error Context** | Input (opt) | Ошибка предыдущей попытки (для retry loop) |
| **Mistral Cloud API** | External | LLM-провайдер для генерации запросов 1С |

---

## Индекс и поиск

**В PoC:** Классический retrieval **отсутствует**. Генерация через prompt engineering:

```
┌─────────────────────────────────────────────────────────────┐
│  Prompt Structure                                           │
├─────────────────────────────────────────────────────────────┤
│  SYSTEM_PROMPT (500 tok)                                    │
│  • Структура базы        │
│  • Правила        │
│  • Пример запроса                                           │
├─────────────────────────────────────────────────────────────┤
│  User Query (50-100 tok)                                    │
│  • Sanitized query (без ФИО)                                │
│  • Error context (если retry)                               │
└─────────────────────────────────────────────────────────────┘
```

## Reranking

**Не применяется в PoC.** Генерация детерминированная (temperature=0.1).


## Ограничения

| Параметр | Значение | Обоснование |
|----------|----------|-------------|
| **Max Input Tokens** | ~8000 | Лимит Mistral |
| **Max Output Tokens** | 500 | Достаточно для запроса 1С |
| **Temperature** | 0.1 | Детерминированная генерация |
| **Latency SLA** | < 10 сек | Ожидаемое: 2-5 сек |
| **Cost** | Бесплатно | Текущий тариф Mistral |
| **Language** | Русский | Поддерживается моделью |
